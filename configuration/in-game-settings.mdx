title: 'In-Game Settings' description: 'Configure your in-game settings for optimal performance with the bot.'

    Windowed Mode: Call of Duty must be in Windowed mode. Press Alt+Enter in-game to toggle.

    Resolution: Use a resolution below 4K (e.g., 1920x1080, 2560x1440).

    Aspect Ratio: Set to Automatic.

    Input Device: Must be set to Keyboard & Mouse.

    Text Size/HUD: Must be set to Default.

    HUD Layout/Bounds: Must be set to Default.

    Shader Quality: Set to Medium or higher.

    Texture Resolution: Set to Normal.

    Parachute Auto-Deploy: Set to Free Fall (Manual).

    Core Modes Only: Do not use Hardcore modes. The bot relies on visual elements that are not present in Hardcore.
